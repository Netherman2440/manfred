# manfred

Minimalny starter projektu LangGraph w Pythonie.

## Wymagania

- Python 3.12+
-  `uv` (`https://docs.astral.sh/uv/`)

## Szybki start (uv)

Linux:
```bash
cd src
uv venv
uv sync 
source .venv/bin/activate
export PYTHONPATH="$PWD"
```



## Testy

```bash
uv run pytest
```

albo:

```bash
pytest
```
